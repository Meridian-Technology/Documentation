# Compass

Compass is Meridian's core module for **room discovery, booking, and space intelligence**. It provides students and staff with a unified platform to find study spaces, manage room bookings, access campus services, and provide real-time feedback on space quality.

## Core Capabilities

### Room & Space Management

- **Real-time Availability**: Live updates on room availability and occupancy
- **Advanced Search**: Filter by capacity, amenities, location, and availability
- **Booking System**: Instant reservations with conflict resolution
- **Recurring Bookings**: Support for regular study sessions and meetings

### Space Intelligence

- **Quality Ratings**: Student feedback on temperature, noise, lighting, and equipment
- **Utilization Analytics**: Track space usage patterns and identify underutilized areas
- **Maintenance Alerts**: Real-time reporting of equipment failures and facility issues
- **Predictive Insights**: Data-driven recommendations for space optimization

### Event Integration

- **Event-Room Linking**: Automatically book spaces when creating events
- **Capacity Planning**: Match event size with appropriate room capacity
- **Resource Management**: Track equipment and amenities needed for events

## Architecture

Compass integrates with:

- **Frontend**: React-based user interface (`Meridian/frontend`)
- **Backend API**: RESTful endpoints for room and event management
- **Real-time Updates**: WebSocket integration for live availability
- **Calendar Integration**: Sync with external calendar systems
- **SIS Integration**: Import classroom data and course schedules

## Use Cases

- **Students**: Find and book study spaces quickly
- **Faculty**: Reserve classrooms and meeting spaces
- **Administrators**: Monitor space utilization and optimize resource allocation
- **Facilities**: Receive real-time maintenance requests and space quality feedback

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (version 19 or higher)
- **npm** (comes with Node.js)
- **Git** for version control
- **PostgreSQL** (for backend development)

You can verify your installations:

```bash
node -v
npm -v
git --version
psql --version
```

## Clone the Repository

Clone the Meridian monorepo to your local machine:

```bash
git clone <repository-url>
cd Meridian-Mono
```

## Installation

Meridian uses a monorepo structure with separate frontend and backend directories. Install dependencies for each:

### 1. Root Dependencies

```bash
npm install
```

### 2. Frontend Dependencies

```bash
cd Meridian/frontend
npm install
```

### 3. Backend Dependencies

```bash
cd ../backend
npm install
```

## Environment Setup

### Backend Configuration

Create a `.env` file in `Meridian/backend/`:

```bash
# Database
DATABASE_URL=postgresql://user:password@localhost:5432/meridian

# Authentication
JWT_SECRET=your-secret-key
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret

# Server
PORT=5001
NODE_ENV=development
```

### Frontend Configuration

Create a `.env` file in `Meridian/frontend/`:

```bash
REACT_APP_API_URL=http://localhost:5001
REACT_APP_GOOGLE_CLIENT_ID=your-google-client-id
```

## Running the Application

### Development Mode (Both Frontend & Backend)

From the root directory, run:

```bash
npm run dev
```

This starts:
- **Frontend**: `http://localhost:3000`
- **Backend API**: `http://localhost:5001`

### Running Separately

**Frontend only:**
```bash
cd Meridian/frontend
npm start
```

**Backend only:**
```bash
cd Meridian/backend
npm start
```

## Database Setup

### Initialize PostgreSQL Database

```bash
# Create database
createdb meridian

# Run migrations
cd Meridian/backend
npm run migrate
```

## Verify Installation

1. **Frontend**: Navigate to `http://localhost:3000` - you should see the Meridian landing page
2. **Backend**: Check `http://localhost:5001/api/health` - should return a health status

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port Already in Use">
    If port 3000 or 5001 is already in use:
    ```bash
    # Kill process on port 3000
    lsof -ti:3000 | xargs kill
    
    # Kill process on port 5001
    lsof -ti:5001 | xargs kill
    ```
  </Accordion>
  
  <Accordion title="Database Connection Issues">
    Ensure PostgreSQL is running:
    ```bash
    # macOS
    brew services start postgresql
    
    # Linux
    sudo systemctl start postgresql
    ```
  </Accordion>
  
  <Accordion title="Module Not Found Errors">
    Clear node_modules and reinstall:
    ```bash
    rm -rf node_modules package-lock.json
    npm install
    ```
  </Accordion>
</AccordionGroup>
